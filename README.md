# ESP News

Site web d'actualités développé dans le cadre du module **Architecture Logicielle**.

## Description

Cette branche contient la version initiale du projet développée selon une approche **procédurale**.

L'application est organisée autour de scripts PHP classiques où la logique métier, l'accès aux données et l'affichage sont gérés directement dans les différentes pages du projet.

Cette approche permet de réaliser rapidement une application fonctionnelle, mais peut rendre le code plus difficile à maintenir et à faire évoluer lorsque le projet devient plus important.
