# ESP News

Site web d'actualités développé dans le cadre du module **Architecture Logicielle**.

## Description

Cette branche contient la version du projet implémentée selon l'architecture **MVC (Model - View - Controller)**.

L'approche MVC permet de séparer l'application en trois composants principaux :

* **Model** : gère les données et les interactions avec la base de données.
* **View** : responsable de l'affichage et de l'interface utilisateur.
* **Controller** : assure le traitement des requêtes et la communication entre les modèles et les vues.

Cette architecture favorise une meilleure organisation du code, facilite sa maintenance et améliore son évolutivité.
